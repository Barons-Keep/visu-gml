# Visu Editor

## Launching the Editor

To enable the editor, open the menu and navigate to:

> `Settings → Developer → Editor: Enable`

To open the editor:

1. Exit the menu by pressing `ESC` until no menu is visible.
2. Press `F5` to open or close the editor.

To exit the editor:

- Press `ESC`, or
- Use the context menu: `File → Exit`

After leaving the editor, the main menu can be opened again by pressing `ESC`.

## Keyboard Shortcuts

Editor mode provides multiple keyboard shortcuts.

---

### General

| Shortcut | Action |
|----------|---------|
| `F1` | Toggle left panel |
| `F2` | Toggle bottom panel |
| `F3` | Toggle right panel |
| `F4` | Toggle track control panel |
| `F5` | Open / close editor |
| `F6` | Toggle **Event Inspector** (left panel) |
| `F7` | Toggle **Template Inspector** (left panel) |
| `F8` | Enable / disable **debug camera (keyboard)** mode (`WASD` to move, `Q` down, `E` up) |
| `F9` | Enable / disable **debug camera (mouse)** mode (`ALT` for mouse look, mouse wheel up/down = camera up/down) |
| `F12` | Show / hide scene debug state |
| `CTRL + N` | Open new project wizard |
| `CTRL + E` | Edit current project |
| `CTRL + O` | Open project |
| `CTRL + S` | Save current project |
| `CTRL + R` | Reload `visu.sfx` from disk |
| `CTRL + L` | Enable / disable autosave |
| `CTRL + U` | Enable / disable updating background services while paused |
| `TAB` | Jump focus to next text field |
| `SHIFT + TAB` | Jump focus to previous text field |

---

### Event Inspector

| Shortcut | Action |
|----------|---------|
| `CTRL + A` | Execute selected event |
| `CTRL + SHIFT + A` | Apply template (save changes) |
| `CTRL + B` | Create a brush from the selected event |

---

### Brush Toolbar

| Shortcut | Action |
|----------|---------|
| `V` | Select tool |
| `E` | Erase tool |
| `B` | Brush tool |
| `C` | Clone tool |
| `Q` | Enable / disable grid snapping |
| `CTRL + D` | Execute brush |
| `CTRL + SHIFT + D` | Apply brush (save changes) |

---

### Track Controls

| Shortcut | Action |
|----------|---------|
| `SPACE` | Play / pause track |
| `CTRL + ,` | Rewind track backward by 5 seconds |
| `CTRL + .` | Rewind track forward by 5 seconds |

---

### Timeline

| Shortcut | Action |
|----------|---------|
| `-` | Timeline zoom out |
| `+` | Timeline zoom in |
| `CTRL + F` | Enable / disable smooth view follow of current timestamp |
| `CTRL + Z` | Undo timestamp action |
| `CTRL + SHIFT + Z` | Redo timestamp action |
| `CTRL + LMB (on event)` | Add or remove event from selection. Marks event as **main** (green frame) |
| `BACKSPACE` / `DEL` | Delete selected event(s) |
